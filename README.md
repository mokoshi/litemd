# litemd

litemd は、Tauri、React、TypeScript で作る軽量なデスクトップ Markdown エディタです。

## 機能

- エディタとプレビューを並べた 2 ペイン表示
- 開いている Markdown ファイルを切り替えられるブラウザ風のタブ
- 編集内容の自動保存
- エディタとプレビューのスクロール同期
- Markdown プレビューでの Mermaid チャート表示
- 現在開いているファイルの Git 差分表示
- 別経路で変更されたファイルの自動再読み込み
- `litemd A.md` のような CLI 起動
- アプリ内からの自動アップデート

## Git 差分表示

Git 管理下のファイルを開いている場合、エディタ上の内容と `HEAD` の差分を確認できます。
Git 管理下の新規ファイルは、空ファイルからの差分として表示します。

## CLI 起動

CLI コマンドをインストールすると、シェルから Markdown ファイルを開けます。

```sh
litemd A.md
litemd A.md B.md
```

すでにアプリが起動している場合、後続の CLI 実行は既存ウィンドウに転送され、
対象ファイルはタブとして開かれます。
